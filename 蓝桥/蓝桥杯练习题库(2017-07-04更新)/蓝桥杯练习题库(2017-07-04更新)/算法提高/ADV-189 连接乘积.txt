算法提高 连接乘积
时间限制：1.0s   内存限制：256.0MB
问题描述　　192这个数很厉害，用它分别乘以1、2、3，会得到：
　　192 x 1 = 192
　　192 x 2 = 384
　　192 x 3 = 576
　　把这三个乘积连起来，得到192384576，正好是一个1~9的全排列
　　我们把上面的运算定义为连接乘积：
　　m x (1 ... n) = k（其中m &gt; 0 且 n &gt; 1，对于上例，m = 192、n = 3、k = 192384576）
　　即k是把m分别乘以1到n的乘积连接起来得到的，则称k为m和n的连接乘积。
　　按字典序输出所有不同的连接乘积k，满足k是1~9的全排列输出格式　　每个k占一行样例输出显然，结果中应包含一行：
192384576
